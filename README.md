Syarat Menjalankan Aplikasi:



* Java Development Kit (JDK) Versi 17 atau lebih tinggi.
* IntelliJ IDEA / Android Studio Versi terbaru dengan plugin Android yang sudah terpasang.
* Android SDK 34 (Android 14).
* Emulator atau Perangkat Fisik Android.



Cara Menjalankan pada Intelij IDEA dan emulator (plugin):

1. Sinkronisasi Aplikasi dengan Graddle (Logo gajah)
2. Buka Device Manager
3. Pastikan sudah ada perangkat yang terdaftar, jika belum bisa ambil model Handphone yang sesuai (misalnya: Pixel 9)
4. Klik ikon Play kecil di sebelah nama perangkat untuk menyalakan ponsel virtual tersebut.
5. Jalankan/run aplikasi pada intelij.
6. Aplikasi akan secara otomatis dibuild dan terbuka pada emulator.
